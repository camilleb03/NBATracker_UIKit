## Technologies
A list of technologies used within the project:
* Xcode 12.4
* Swift 5
* UIKit
* data.nba.net API
* cdn.nba.net API

## Features

A few of the things you can do with NBATracker:

* View live scoreboards of the current games
* View the league standings by conference
